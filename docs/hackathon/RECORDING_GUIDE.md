# 플레이 영상 촬영 가이드 (백층 던전)

> NAN 2026 제출물 2 — 30~60초 실플레이 영상 (YouTube)
> ⚠️ **규정: 실제 플레이 화면 그대로. AI 합성/조작·타인 영상 도용 금지.** 반드시 본인이 직접 플레이한 화면을 녹화할 것. **소리 포함 필수** (효과음·BGM이 전부 코드 합성 — 강점).

## 0. 촬영 전 준비 (30초)

1. **소리 켜기** — 게임 우상단 🔊 버튼이 켜져 있는지 확인 (음소거면 효과음·BGM이 안 들림).
2. **연습 1판** — 아래 대본대로 한 번 미리 돌려보면 실제 촬영이 매끄럽습니다.
3. (선택) 지름길: 주소창에 `https://hakhyun-kim.github.io/dungeon100/?debug` 로 열면 게임 입장 후 **Shift+D**로 원하는 층(10층 보스, 14/28/42/49/56층 소녀 흔적 등)으로 점프 가능 — "보여주고 싶은 장면"만 빠르게 담을 때 유용. 단, **녹화 화면에 디버그 창은 나오지 않게** (Esc로 닫고 나서 녹화).

## 1. 촬영 대본 (총 ~50초)

| 시간 | 장면 | 담을 것 |
|------|------|---------|
| 0–6초 | **인트로 + 마을** | 책이 "7 × 8 = ?"을 묻는 인터랙티브 인트로 한 컷 → 촌장 대화 → 던전 입구 난이도 선택(🎒/🧠) |
| 6–14초 | **던전 전투** | 자동 조준 사격, 몬스터 처치(💥 파티클·데미지 숫자), 나침반 화살표로 이동 |
| 14–30초 | **보물상자 = 두 문 달리기** ⭐핵심 | 상자 접촉 → 트랙 달리기 → 정답 문 몸으로 통과 → "더 달린다!" 선택 → 전설의 보물 |
| 30–40초 | **되찾은 기억 or 보스** | 보물 후 회상 씬("그때 그 라면") 한 컷 / 또는 10층 보스 "페이지의 수호자" 전투 |
| 40–50초 | **다음 층 + 여운** | 포털 → 벽의 글귀(스토리) → 더 깊은 층의 달라진 몬스터. 마지막 1~2초는 타이틀 로고나 층수 HUD로 마무리 |

**팁:** 제일 인상적인 "두 문 달리기 통과 → 보물 획득"(14–30초)을 반드시 넣으세요. 이 게임의 정체성이자 심사 포인트입니다.

## 2. 녹화 방법

### PC (권장 — 화질 안정적)
- **Windows 게임바**: `Win + G` → 녹화 버튼(또는 `Win + Alt + R`로 바로 시작/정지). 시스템 소리 포함됨. 저장 위치: `영상\캡처`.
- **크롬 창만 깔끔하게**: 게임을 크롬에서 열고 `F11`(전체화면) 후 녹화하면 주소창·탭이 안 보임.
- (더 좋은 화질/편집) **OBS Studio** 무료 — 장면에 크롬 창만 추가해 녹화.

### 모바일 (세로 화면 느낌을 살리고 싶을 때)
- **아이폰**: 설정 → 제어센터 → 화면 기록 추가 → 제어센터에서 ⏺ (마이크 끄고, 게임 소리만).
- **안드로이드**: 빠른 설정 → 화면 녹화 → "미디어 소리" 켜기.
