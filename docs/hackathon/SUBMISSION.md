## 플레이 영상 촬영 가이드 (30~60초)

- AI 합성·편집 조작 불가 — **실제 플레이 화면 그대로** 녹화. **소리 포함 필수** (효과음이 전부 코드 합성이라는 점이 특징).
- 추천 구성 (총 ~55초):
  1. (0–6초) 인트로 스토리 1~2장 + 마을 촌장 대화 한 컷 (세계관 어필)
  2. (6–12초) 던전 입구 난이도 선택 (🎒/🧠) → 입장
  3. (12–30초) 던전 전투 — 몬스터 처치·파티클·아이템 빌드업
  4. (30–45초) **보물상자 → 두 문 달리기** 통과 → "더 달린다!" → 전설의 보물 or 회상 씬 한 컷
  5. (45–55초) 포털로 다음 층 → 벽의 글귀 → 더 깊은 층의 달라진 몬스터
- YouTube에 **공개 또는 일부 공개(링크 공유)**로 업로드 → 심사 종료까지 유지.
