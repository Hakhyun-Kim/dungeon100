# 백층 던전 (Dungeon 100)

『백층 던전의 비밀』을 읽다 잠든 2026년의 대학생 — 눈을 떠 보니 책 속 마을.
집으로 돌아가는 유일한 길, 매판 새로 만들어지는 던전을 뚫고 **100층까지 내려가는 3D 로그라이크**.
**NAN 2026 (NHN Game × AI Hackathon) 사전과제 제출작**입니다. 자매작 「두 문 러너」의 두 문 달리기가 이 게임의 보물상자 미니게임으로 **통합되어 있습니다**.

> ### 🎮 바로 플레이: https://hakhyun-kim.github.io/dungeon100/
> 설치·로그인 없이 브라우저(모바일/PC)에서 바로 실행됩니다.
> ### 🎬 자동 시연 (클릭 한 번, 약 90초): https://hakhyun-kim.github.io/dungeon100/?demo
> 게임이 스스로 마을·전투·미니게임 2종·깊은 층 테마·보스까지 자막과 함께 보여줍니다.

## 게임 방법

- **걸어다니는 3D 마을**에서 시작 — NPC(촌장·니나·무크)와 대화하고, 던전 입구에서 던전 종류를 고르세요: **🎒 초등학교 던전**(쉬운 문제) / **🧠 어른 던전**(어려운 문제) / **👹 몬스터 던전**(문제 대신 전투).
- 층마다 **절차 생성**되는 방과 복도를 탐험하며 보라색 포털을 찾아 다음 층으로. 10층마다 보스 **"페이지의 수호자"**가 포털을 봉인하고, 그 외 층은 진홍빛 **수문장**이 문 앞을 지킵니다.
- 이동 — 모바일: 화면 아무 곳이나 드래그(가상 스틱) · PC: `WASD`/방향키. 공격은 가까운 적 **자동 조준**.
- 🗝️ **보물상자 = 미니게임** (던전 종류에 따라):
  - 수학 던전 → **두 문 달리기**: 트랙을 달리며 정답이 적힌 문을 몸으로 통과 (자매작 [두 문 러너](https://hakhyun-kim.github.io/door-runner/)의 인게임 재현). 문마다 "여기서 받기 vs 더 달리기" — **3연속 완주 = 전설의 보물**, 틀리면 전부 빈손!
  - 몬스터 던전 → **몬스터 아레나**: 몰려오는 무리를 뚫고 보석 3개를 주우면 같은 보상. 쓰러져도 본체는 무사 — 몇 번이고 재도전.
- 층을 돌파할 때마다 보상 카드 3택 1로 빌드를 쌓고(공격력·연사·멀티샷·이속·체력·사거리), 처치 코인으로 마을 **대장간에서 영구 강화**(죽어도 유지, 무한 단련)를 삽니다.
- 💭 보물을 얻으면 **잊고 있던 집의 기억**이 돌아오고, 층마다 **벽의 글귀**가 던전의 비밀을 속삭입니다. 5층마다 나타나는 문으로 마을에 들르면 **부활 지점**이 갱신되고 — 깊이 내려갈수록 **마을에도 시간이 흐릅니다** (가을 → 습격 → 방벽 → 폐허와 새벽).
- 깊이 내려갈수록 몬스터·던전 색·안개도 변하고 짙어집니다. 체력이 0이 되면 마지막 마을에서 부활 — 100층 끝의 황금 문에는 **엔딩**(2종 + 에필로그)이 기다립니다.

## 로컬 실행

```bash
npm install
npm run dev   # http://localhost:5175
```
요구 사항: Node.js 20+. `npm run build` 프로덕션 빌드, `npm run typecheck` 타입 검사.

## 원작 — 두 문 러너 (Two Door Runner)

보물상자 미니게임 "두 문 달리기"는 자매작 **두 문 러너**(초등 수학 러너, **마찬가지로 AI(Claude Code)로 제작**)의 핵심 메커닉을 이 게임에 **통합**한 것입니다. 두 문 러너의 "정답 문을 몸으로 열며 달리는" 경험이 백층 던전 안에서 그대로 재현됩니다.

- 🎮 두 문 러너 플레이: https://hakhyun-kim.github.io/door-runner/
- 📦 두 문 러너 저장소: https://github.com/Hakhyun-Kim/door-runner
